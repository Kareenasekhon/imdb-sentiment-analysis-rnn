# Sentiment Analysis on IMDB Reviews using RNN

This project performs sentiment analysis on IMDB movie reviews using a Recurrent Neural Network (RNN) built with PyTorch.

## 📌 Overview
The model classifies reviews as:
- Positive
- Negative

## ⚙️ Workflow
- Data preprocessing (cleaning, stopwords, stemming)
- TF-IDF vectorization
- Train-test split
- RNN model training
- Evaluation and prediction

## 🧠 Model
- RNN (Recurrent Neural Network)
- Binary Classification

## 📊 Dataset
- IMDB dataset (50,000 reviews)

## ▶️ How to Run
1. Open the notebook
2. Run all cells step by step

## 💡 Example
Input: This movie was amazing  
Output: Positive Review
